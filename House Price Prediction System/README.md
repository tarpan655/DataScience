# House Price Prediction System

Starter repository for a house price prediction project. Contains data, notebooks, and starter code to preprocess data, engineer features, train models, and save artifacts.

Structure:
- notebooks/: EDA and modeling notebooks
- data/: raw and processed data placeholders
- src/: reusable Python modules
- models/: trained model artifacts

How to get started:
1. Create a Python virtual environment and install dependencies: `pip install -r requirements.txt`
2. Place raw data CSV files into `House Price Prediction System/data/raw/`
3. Run the notebooks or use the scripts in `House Price Prediction System/src/`
